<td><p><strong>Note d’application</strong></p></td>
<td class="en">
<p>This class comprises activities that result in instances of F2 Expression coming into existence. An instance of F2 Expression is considered to be created when it is captured on a carrier other than the creator’s brain.</p>
<p>Although F2 Expression is an abstract entity, a conceptual object, the creation of an expression inevitably also affects the physical world: when you scribble the first draft of a poem on a sheet of paper, you produce an instance of F3 Manifestation and an instance of F5 Item. F28 Expression Creation is a subclass of E12 Production because the recording of the expression causes a physical modification of the E18 Physical Thing that serves as the carrier. The creation of an instance of F2 Expression coincides with the creation of the first instance of F3 Manifestation that R4 embodies (is embodied in) this instance of F2 Expression.</p>
<p>The P2 has type (is type of) property can be used to specify the type of the instance of F28 Expression Creation (i.e., activities such as translating, revising, or arranging music are types of creation process). The type of the process is distinct from the type of result even though the typology frequently used for instances of the resulting F2 Expressions may imply the category of the instance of the F28 Expression Creation.</p>
<p>An instance of F28 Expression Creation may use as source material one or more specific instances of F2 Expression. When the source expression is documented this is also expressed by the property R76 is derivative of (has derivative).</p>
</td>
<td>
<p>Cette classe comprend les activités dont le résultat est de faire exister des instances de <code class="language-plaintext highlighter-rouge">F2_Expression</code>. Une instance de <code class="language-plaintext highlighter-rouge">F2_Expression</code> est considérée comme créée lorsqu’elle est capturée sur un support autre que le cerveau de son créateur.</p>
<p>Bien que <code class="language-plaintext highlighter-rouge">F2_Expression</code> soit une entité abstraite, un objet conceptuel, la création d’une expression affecte inévitablement aussi le monde physique : lorsque vous griffonnez le premier brouillon d’un poème sur une feuille de papier, vous produisez une instance de <code class="language-plaintext highlighter-rouge">F3_Manifestation</code> et une instance de <code class="language-plaintext highlighter-rouge">F5_Item</code>. <code class="language-plaintext highlighter-rouge">F28_Création_d’expression</code> est une sous-classe de <code class="language-plaintext highlighter-rouge">E12_Production</code>, car l’enregistrement de l’expression provoque une modification de <code class="language-plaintext highlighter-rouge">E18_Chose_matérielle</code> qui sert de support. La création d’une instance de <code class="language-plaintext highlighter-rouge">F2_Expression</code> coïncide avec la création de la première instance de <code class="language-plaintext highlighter-rouge">F3_Manifestation</code> qui incarne (<code class="language-plaintext highlighter-rouge">R4_incarne</code>) cette instance de <code class="language-plaintext highlighter-rouge">F2_Expression</code>.</p>
<p>La propriété <code class="language-plaintext highlighter-rouge">P2_a_pour_type (est_le_type_de)</code> peut être utilisée pour préciser le type d’instance de <code class="language-plaintext highlighter-rouge">F28_Création_d’expression</code> (p. ex. des activités telles que la traduction, la révision ou l’arrangement de la musique sont des types de processus de création). Le type de processus est distinct du type de résultat même si la typologie fréquemment utilisée pour les instances des expressions résultantes peut impliquer la catégorie de l’instance de la création d’expression (<code class="language-plaintext highlighter-rouge">F28_Création_d’expression</code>). </p>
<p>Une instance de création d’expression (<code class="language-plaintext highlighter-rouge">F28_Création_d’expression</code>) peut utiliser comme source une ou plusieurs instances spécifiques de l’expression (<code class="language-plaintext highlighter-rouge">F2_Expression</code>). Lorsque l’expression source est documentée, elle est également exprimée par la propriété <code class="language-plaintext highlighter-rouge">R76_est_dérivé_de (a_pour_dérivé</code>).</p>
</td>
