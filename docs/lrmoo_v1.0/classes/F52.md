<td><p><strong>Note d’application</strong></p></td>
<td class="en">
<p>This class comprises periods of continuous use of a specific instance of E41 Appellation for a particular instance of E1 CRM Entity by an E39 Actor. It includes in particular the use of the name by its carrier. Characteristically, actors performing an activity may choose a particular appellation for themselves in the context of this activity. Such cases should be modelled by additionally classifying these activities as instances of F52 Name Use Activity.</p>
<p>It is possible to specify the type of name use, through the <em>P2 has type (is type of)</em> property, e.g.: use of a pseudonym, use of a married name, use of a birth name, use of a blended name, use of a religious name.</p>
</td>
<td>
<p>Cette classe comprend des périodes d’utilisation continue, par une instance de <code class="language-plaintext highlighter-rouge">E39_Actant</code>, d’une instance spécifique de <code class="language-plaintext highlighter-rouge">E41_Appellation</code> pour une instance spécifique de <code class="language-plaintext highlighter-rouge">E1_Entité_CRM</code>. Elle comprend notamment l’usage d’un nom particulier par celui.celle qui le porte. Il est caractéristique des actants performant l’activité de choisir un nom spécifique dans le contexte de cette activité. De tels cas doivent être modélisés en classifiant ces activités non seulement comme des instances de <code class="language-plaintext highlighter-rouge">E7_Activité</code>, mais aussi comme des instances de <code class="language-plaintext highlighter-rouge">F52_Activité_d’utilisation_du_nom</code>. </p>
<p>Il est possible de préciser le type d’activité d’utilisation du nom (p. ex. pseudonyme, nom marital, nom de naissance, nom composé, nom religieux) par l’usage de la propriété <code class="language-plaintext highlighter-rouge">P2_a_pour_type (est_le_type_de)</code>. </p>
</td>
</tr>

</li>
<li><p>using the name ‘Charles Dodgson’ when authoring works of mathematics and logics (<em>has type</em> Use of a birth name (E55))</p>
</li>

</li>

</li>
