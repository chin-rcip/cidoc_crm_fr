<td><p><strong>Note d’application</strong></p></td>
<td class="en">
<p>This property associates an instance of F18 Serial Work with the instance of E29 Design or Procedure that specifies the issuing policy planned by this instance of F18 Serial Work, such as sequencing pattern, expected frequency and expected regularity.</p>
</td>
<td>
<p>Cette propriété associe une instance de <code class="language-plaintext highlighter-rouge">F18_Oeuvre_de_publication_en_série</code> avec l’instance de <code class="language-plaintext highlighter-rouge">E29_Conceptualisation_ou_procédure</code> qui spécifie la politique d’émission prévue par cette instance de <code class="language-plaintext highlighter-rouge">F18_Oeuvre_de_publication_en_série</code>, telle que le schéma de séquençage, la fréquence attendue et la régularité attendue.</p>
<p>Cette propriété comprend la note de l’éditeur suivante : Transférer cette propriété à PRESSOO une fois que la version  2.0 aura un projet stable. Mettre en œuvre cette propriété uniquement en conjonction avec une implémentation de PRESSOO.</p>
