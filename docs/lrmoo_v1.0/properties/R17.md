<td><p><strong>Note d’application</strong></p></td>
<td class="en">
<p>This property associates an instance of F2 Expression that was externalised during a particular instance of F28 Expression Creation event with that particular creation event. An instance of expression creation creates an instance of expression and also creates any expressions that are parts of that expression.</p>
</td>
<td>
<p>Cette propriété associe une instance de <code class="language-plaintext highlighter-rouge">F2_Expression</code> qui a été exteriorisée lors d’une instance particulière de l’évènement de <code class="language-plaintext highlighter-rouge">F28_Création_d'expression</code> avec cet évènement de création spécifique. Une instance de création d’expression crée une instance d'expression et toutes les expressions qui en font partie.</p>
