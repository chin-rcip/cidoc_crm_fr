<td><p><strong>Note d’application</strong></p></td>
<td class="en">
<p>This property associates an instance of F1 Work which modifies the content of another instance of F1 Work with the latter. This property is transitive, asymmetric and irreflexive. The inverse of this property is equivalent to the developed path: F1 Work(1). R3 is realised in: F2 Expression(1). P16i was used for: F28 Expression Creation. R17 created: F2 Expression(2). R3i realises: F1 Work(2). That is, F1 Work(1). R2i has derivative: F1 Work (2), without needing to specify the specific expressions involved in the derivation.</p>
</td>
<td>
<p>Cette propriété associe une instance de <code class="language-plaintext highlighter-rouge">F1_Œuvre</code> à une autre instance de <code class="language-plaintext highlighter-rouge">F1_Œuvre</code> qui modifie le contenu de cette dernière. </p>
<p>Cette propriété est transitive, asymétrique et irréflexive.</p>
<p>La propriété inverse est l’équivalent du chemin sémantique complètement développé <code class="language-plaintext highlighter-rouge">F1_Œuvre</code> [1], <code class="language-plaintext highlighter-rouge">R3_est_réalisé_dans</code>, <code class="language-plaintext highlighter-rouge">F2_Expression</code> [1], <code class="language-plaintext highlighter-rouge">P16i_a_été_mobilisé_pour</code>, <code class="language-plaintext highlighter-rouge">F28_Création_d’expression</code>, <code class="language-plaintext highlighter-rouge">R17_a_créé</code>, <code class="language-plaintext highlighter-rouge">F2_Expression</code> [2], <code class="language-plaintext highlighter-rouge">R3i_réalise</code>, <code class="language-plaintext highlighter-rouge">F1_Œuvre</code> [2], c.-à-d. <code class="language-plaintext highlighter-rouge">F1_Œuvre</code> [1], <code class="language-plaintext highlighter-rouge">R2i_a_pour_dérivé</code>, <code class="language-plaintext highlighter-rouge">F1_Œuvre</code> [2], sans avoir à indiquer les expressions spécifiques impliquées dans la dérivation.</p>
