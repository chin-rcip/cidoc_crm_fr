<td><p><strong>Note d’application</strong></p></td>
<td class="en">
<p>This property associates an instance of F32 Item Production Event with the set of signs provided by the publisher to be carried by all of the produced items (i.e., the instances of F5 Item) and any other physical features foreseen as integral to the instance of F3 Manifestation that is materialised.</p>
</td>
<td>
<p>Cette propriété associe une instance de <code class="language-plaintext highlighter-rouge">F32_Évènement_de_production_d'item</code> à l’ensemble des signes fournis par l’éditeur devant être portés par tous les items produits (c.-à-d. les instances de <code class="language-plaintext highlighter-rouge">F5_Item</code>) et à toute autre caractéristique physique prévue comme faisant partie intégrante de l’instance de <code class="language-plaintext highlighter-rouge">F3_Manifestation</code> qui est matérialisée.</p>
</td>
