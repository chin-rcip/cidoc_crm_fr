<td><p><strong>Note d’application</strong></p></td>
<td class="en">
<p>This property associates an instance of F12 Nomen with another instance of F12 Nomen which co-refers to the same instance of E1 CRM Entity. This property is not transitive. It is symmetric and irreflexive.</p>
</td>
<td>
<p>Cette propriété associe une instance de <code class="language-plaintext highlighter-rouge">F12_Nomen</code> à une autre instance de <code class="language-plaintext highlighter-rouge">F12_Nomen</code> qui font toutes deux référence à la même instance de <code class="language-plaintext highlighter-rouge">E1_Entité_CRM</code>. Cette propriété n’est pas transitive. Elle est symétrique et réflexive.</p>
</td>
</tr>
<tr>
<td><p><strong>Propriétés</strong></p></td>
<td class="en">
<p>R56.1 has type: E55 Type</p>
<p>This property allows for specifying the particular kind of relationship that holds between the domain nomen and the range nomen, such as by being a derivation, an alternative, a lexical variant, etc. Typing the association may cause loss of symmetry</p>
</td>
<td>
<p><code class="language-plaintext highlighter-rouge">R56.1_a_pour_type</code> : <code class="language-plaintext highlighter-rouge">E55_Type</code> </p>
<p>Cette propriété permet de préciser le type particulier de relation qui existe entre le nomen du domaine et le nomen de la portée, par exemple en tant que dérivation, alternative, variante lexicale, etc. Le typage de l’association peut entraîner une perte de symétrie.</p>
</td>
</tr>
