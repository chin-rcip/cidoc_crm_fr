<td><p><strong>Note d’application</strong></p></td>
<td class="en">
<p>This property associates an instance of F38 Character with an instance of E39 Actor that the character is motivated by or is intended to represent. An instance of F38 Character may be based on a combination of features taken from several actors. This property is a shortcut of the more fully developed path from E28 Conceptual Object, restricted to F38 Character, through the inverse of <em>P94 has created (was created by):</em> E65 Creation. <em>P17 was motivated by (motivated)</em> to E1 CRM Entity restricted to E39 Actor.</p>
</td>
<td>
<p>Cette propriété associe une instance de <code class="language-plaintext highlighter-rouge">F38_Actant_fictif</code> à une instance de <code class="language-plaintext highlighter-rouge">E39_Actant</code> où celle-ci est l’inspiration derrière l’actant fictif ou est censée être représentée par l’actant fictif. Une instance de <code class="language-plaintext highlighter-rouge">F38_Actant_fictif</code> peut être fondée sur une combinaison de caractéristiques provenant de plusieurs actants. Cette propriété est un raccourci du chemin plus développé de <code class="language-plaintext highlighter-rouge">E28_Objet_conceptuel</code>, restreint à <code class="language-plaintext highlighter-rouge">F38_Actant_fictif</code>, <code class="language-plaintext highlighter-rouge">P94i_a_été_créé_par</code>, <code class="language-plaintext highlighter-rouge">E65_Création</code>, <code class="language-plaintext highlighter-rouge">P17_a_été_motivé_par (a_motivé)</code>, <code class="language-plaintext highlighter-rouge">E1_Entité_CRM</code> restreinte à <code class="language-plaintext highlighter-rouge">E39_Actant</code>.</p>
</td>
