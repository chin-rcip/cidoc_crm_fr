<td><p><strong>Note d’application</strong></p></td>
<td class="en">
<p>This property associates an instance of F1 Work with another instance of F1 Work whose content was inspired by that instance of F1 Work. The content of the range work instance served in some way as a source of ideas for the domain work instance. Neither instance of F1 Work may be a part of the other. This property is not transitive. It is irreflexive.</p>
</td>
<td>
<p>Cette propriété associe une instance de <code class="language-plaintext highlighter-rouge">F1_Œuvre</code> à une autre instance de <code class="language-plaintext highlighter-rouge">F1_Œuvre</code> dont le contenu a été inspiré par la première. Le contenu de l’instance de portée de <code class="language-plaintext highlighter-rouge">F1_Œuvre</code> a servi d’une quelconque manière de source d’idées pour l’instance de domaine de <code class="language-plaintext highlighter-rouge">F1_Œuvre</code>. Aucune des deux instances de <code class="language-plaintext highlighter-rouge">F1_Œuvre</code> ne peuvent faire partie de l’autre. Cette propriété n’est pas transitive. Elle est irréflexive.</p>
</td>
