<td><p><strong>Note d’application</strong></p></td>
<td class="en">
<p>This property associates an instance of F3 Manifestation with an instance of E55 Type describing the kind of physical form that characterizes instances of F5 Item carrying this F3 Manifestation.</p>
<p>In the case of instances of manifestations intended to be rendered by mediation (such as with electronic devices), the form also indicates the kind of equipment and software tools necessary.</p>
</td>
<td>
<p>Cette propriété associe une instance de <code class="language-plaintext highlighter-rouge">F3_Manifestation</code> à une instance de <code class="language-plaintext highlighter-rouge">E55_Type</code> qui décrit le type de forme physique caractérisant les instances de <code class="language-plaintext highlighter-rouge">F5_Item</code> supportant cette <code class="language-plaintext highlighter-rouge">F3_Manifestation</code>. Dans le cas des manifestations conçues pour une lecture à l’aide d’un support (comme des appareils électroniques), la forme indique aussi le type d’équipement et les outils logiciels nécessaires.</p>
</td>
