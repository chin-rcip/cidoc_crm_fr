<td><p><strong>Note d’application</strong></p></td>
<td class="en">
<p>This property associates an instance of F1 Work with instances of F2 Expression that bear attributes which are used to characterize the work. The instance of F2 Expression in question must be one that realizes that instance of F1 Work. An instance of F2 Expression may or may not serve as source of representative expression attributes for its associated instance of F1 Work. If it does, it may provide one attribute or many. Only one or more than one of the instances of F2 Expression associated with an instance of F1 Work may provide attributes that characterize that instance of F1 Work. The representative expression attributes can all come from the same expression or from more than one expressions.</p>
</td>
<td>
<p>Cette propriété associe une instance de <code class="language-plaintext highlighter-rouge">F1_Œuvre</code> à des instances de <code class="language-plaintext highlighter-rouge">F2_Expression</code> qui portent des attributs utilisés pour caractériser l’œuvre. L’instance de <code class="language-plaintext highlighter-rouge">F2_Expression</code> en question doit être celle qui réalise cette instance de <code class="language-plaintext highlighter-rouge">F1_Œuvre</code>. Une instance de <code class="language-plaintext highlighter-rouge">F2_Expression</code> peut ou non servir de source d’attributs d’expression représentatifs pour son instance associée de <code class="language-plaintext highlighter-rouge">F1_Œuvre</code>. Si c’est le cas, elle peut fournir un ou plusieurs attributs. Seule une ou plusieurs des instances de <code class="language-plaintext highlighter-rouge">F2_Expression</code> associées à une instance de <code class="language-plaintext highlighter-rouge">F1_Œuvre</code> peuvent fournir des attributs qui caractérisent cette instance de <code class="language-plaintext highlighter-rouge">F1_Œuvre</code>. Les attributs d’expression représentatifs peuvent tous provenir de la même expression ou de plusieurs expressions.</p>
</td>
