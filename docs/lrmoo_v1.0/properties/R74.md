<td><p><strong>Note d’application</strong></p></td>
<td class="en">
<p>This property associates an instance of F1 Work with another instance of F1 Work where all expressions of the former will include some expression of the latter. This property is not transitive. It is irreflexive. This property represents the generalized relationship between works that is described at the expression level using <em>R75 incorporates (is incorporated in)</em>.</p>
</td>
<td>
<p>Cette propriété associe une instance de <code class="language-plaintext highlighter-rouge">F1_Œuvre</code> à une autre instance de <code class="language-plaintext highlighter-rouge">F1_Œuvre</code>, où toutes les expressions de la première incluent des expressions de la seconde. </p>
<p>Cette propriété représente la relation généralisée entre des instances de <code class="language-plaintext highlighter-rouge">F1_Œuvre</code> telle qu’elle est décrite au niveau des expressions par l’usage de <code class="language-plaintext highlighter-rouge">R75_incorpore (est_inclus_dans)</code>. </p>
<p>Cette propriété est non transitive et irréflexive. </p>
</td>
