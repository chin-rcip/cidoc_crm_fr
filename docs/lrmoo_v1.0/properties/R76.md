<td><p><strong>Note d’application</strong></p></td>
<td class="en">
<p>This property associates an instance of F2 Expression with another instance of F2 Expression which was its source or one of its sources. This property is not transitive. It is asymmetric and irreflexive. </p>
<p>This property is also a shortcut of the fully developed path: F2 Expression (1). <em>P16i was used for</em>: F28 Expression Creation. <em>R17 created</em>: F2 Expression (2).</p>
</td>
<td>
<p>Cette propriété associe une instance de <code class="language-plaintext highlighter-rouge">F2_Expression</code> à une autre instance de <code class="language-plaintext highlighter-rouge">F2_Expression</code> qui était sa source ou l’une de ses sources. Cette propriété n’est pas transitive. Elle est asymétrique et irréflexive.</p>
<p><code class="language-plaintext highlighter-rouge">R76_est_dérivé_de (a_pour_dérivé)</code> est un raccourci du chemin sémantique suivant : <code class="language-plaintext highlighter-rouge">F2_Expression</code> [1], <code class="language-plaintext highlighter-rouge">P16i_a_été_mobilisé_pour</code>, <code class="language-plaintext highlighter-rouge">F28_Création_d’expression</code>, <code class="language-plaintext highlighter-rouge">R17_a_créé</code>, <code class="language-plaintext highlighter-rouge">F2_Expression</code> [2].  </p>
