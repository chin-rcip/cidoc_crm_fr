<td><p><strong>Note d’application</strong></p></td>
<td class="en">
<p>This property associates one instance of F1 Work with another instance of F1 Work which is intended to accompany it or to function as a complement for it. This property is neither transitive nor intransitive. It is generally not symmetric and it is irreflexive.</p>
<p>In many but not all cases, one of the instances of F1 Work is primary and can be used without the other work, while the other is secondary and depends on the first work (such as a work that is a concordance for another work).</p>
<p>In some cases a work may have been created to accompany or complement a specific expression of another work. The link to the source expression can be indicated using the property <em>P16 used specific object (was used for)</em> using the path: F1 Work(1). <em>R3 is realised in:</em> F2 Expression(1). <em>P16i was used for:</em> F27 Work Creation. <em>R16 created</em>: F1 Work(2).</p>
</td>
<td>
<p>Cette propriété associe une instance de <code class="language-plaintext highlighter-rouge">F1_Œuvre</code> à une autre instance de <code class="language-plaintext highlighter-rouge">F1_Œuvre</code>, laquelle est vouée à l’accompagner ou à la complémenter. </p>
<p>Cette propriété n’est ni transitive ni non transitive ; elle n’est habituellement pas symétrique et est irréflexive. </p>
<p>Dans plusieurs cas, quoique pas toujours, l’une des instances de <code class="language-plaintext highlighter-rouge">F1_Œuvre</code> est de nature primaire et peut être utilisée sans s’appuyer sur celle dont la nature est secondaire, alors que cette dernière dépend de la première (p. ex. une œuvre qui est la concordance d’une autre). </p>
<p>Dans certains cas, une œuvre peut avoir été créée pour accompagner ou complémenter une expression spécifique d’une autre œuvre. Le lien vers l’expression source peut être indiqué par le biais de la propriété <code class="language-plaintext highlighter-rouge">P16_a_mobilisé_l’objet_spécifique (a_été_mobilisé_pour)</code> en utilisant le chemin sémantique suivant : <code class="language-plaintext highlighter-rouge">F1_Œuvre</code> [1], <code class="language-plaintext highlighter-rouge">R3_est_réalisé_dans</code>, <code class="language-plaintext highlighter-rouge">F2_Expression</code>, <code class="language-plaintext highlighter-rouge">P16i_a_été_mobilisé_pour</code>, <code class="language-plaintext highlighter-rouge">F27_Création_d’œuvre</code>. <code class="language-plaintext highlighter-rouge">R16_a_créé</code>, <code class="language-plaintext highlighter-rouge">F1_Œuvre</code> [2].</p>
</td>
