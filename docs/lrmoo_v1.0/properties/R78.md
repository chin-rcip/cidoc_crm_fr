<td><p><strong>Note d’application</strong></p></td>
<td class="en">
<p>This property associates one instance of F3 Manifestation with another instance of F3 Manifestation that exemplifies the same instance of F2 Expression, when the two instances of F3 Manifestation can be used as alternatives for each other in particular use cases. This property is transitive and symmetric. It is irreflexive.</p>
<p>The alternative manifestations may be in the same physical form, for example, simultaneous publications in different markets. More frequently, the alternative relationship is established when the alternative manifestations are in different physical forms, designed to enable use of the same content with different playback equipment (such as a DVD and Blu-ray disc version of the same videorecording).</p>
</td>
<td>
<p>Cette propriété associe une instance de <code class="language-plaintext highlighter-rouge">F3_Manifestation</code> à une autre instance de <code class="language-plaintext highlighter-rouge">F3_Manifestation</code> qui exemplifie la même instance de <code class="language-plaintext highlighter-rouge">F2_Expression</code>, lorsque les deux instances de <code class="language-plaintext highlighter-rouge">F3_Manifestation</code> peuvent être utilisées comme des représentations alternatives dans certains cas d’utilisation. Cette propriété est transitive, symétrique et irréflexive.</p>
<p>Les manifestations alternatives peuvent être de même format (p. ex. des publications simultanées sur différents marchés). Le plus souvent, cette relation est établie lorsque les manifestations alternatives sont dans des formats différents, conçues pour permettre l’utilisation du même contenu avec différents équipements de lecture (tels qu’un enregistrement vidéo en version DVD et en version Blu-ray).</p>
</td>
