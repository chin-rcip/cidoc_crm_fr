<td><p><strong>Note d’application</strong></p></td>
<td class="en">
<p>This property associates an instance of F1 Work with an instance of E55 Type that describes a category of attribute that is considered essential in characterizing instances of F1 Work. The types of interest will vary depending on the kind of work.</p>
<p>The value of the attribute is considered representative of the instance of F1 Work. It is normally inferred from the values attributed to instances of F2 Expression that realize the work and that are considered canonical or best representative of the work. The values may also be assigned from characteristics abstracted from a more or less nebulous network of similar expressions.</p>
<p>There is no requirement to precisely identify one or more expressions which serve as sources for the values of the types serving as representative expression attributes; however, if this is known, an instance of F2 Expression considered representative of an F1 Work may be related to the instance of F1 Work using the R73 takes representative attribute from (bears representative</p>
<p>attribute for) property.</p>
</td>
<td>
<p>Cette propriété associe une instance de <code class="language-plaintext highlighter-rouge">F1_Œuvre</code> à une instance de <code class="language-plaintext highlighter-rouge">E55_Type</code>, qui décrit une catégorie d’attributs considérés comme essentiels pour caractériser les instances de <code class="language-plaintext highlighter-rouge">F1_Œuvre</code>. Les types d’intérêts varient en fonction du genre de l'œuvre. </p>
<p>La valeur de cet attribut est considérée comme représentative de l’instance de <code class="language-plaintext highlighter-rouge">F1_Œuvre</code>. Elle est généralement déduite des valeurs attribuées aux instances de <code class="language-plaintext highlighter-rouge">F2_Expression</code> qui réalisent l’œuvre et sont jugées comme étant canoniques ou les meilleures représentations de celle-ci. Ces valeurs peuvent également être assignées à partir de caractéristiques abstraites issues d’un réseau plus ou moins nébuleux d’expressions similaires. Il n’est pas nécessaire d’identifier précisément une ou plusieurs expressions servant de sources aux valeurs des types agissant comme attributs d’expression représentatifs. Toutefois, si une telle information est disponible, une instance de <code class="language-plaintext highlighter-rouge">F2_Expression</code> considérée comme représentative d’une <code class="language-plaintext highlighter-rouge">F1_Œuvre</code> peut être reliée à l’instance de <code class="language-plaintext highlighter-rouge">F1_Œuvre</code> à l’aide de la propriété <code class="language-plaintext highlighter-rouge">R73_prend_l’attribut_représentatif_de (porte_l’attribut_représentatif_pour)</code>. </p>
</td>
